# Phantom Solana DeFi Dashboard

A professional, trust-inspiring DeFi portfolio dashboard designed for the Solana ecosystem. Connect your Phantom Wallet and start earning through staking, yield farming, and token swaps.

## Features

- **Portfolio Dashboard** - Track your SOL and SPL token balances with beautiful charts
- **SOL Staking** - Compare validators, calculate rewards, and stake your SOL
- **Yield Farming** - Discover DeFi pools with APY comparison and risk indicators
- **Token Swap** - Jupiter-style swap interface with real-time price charts
- **Transaction History** - Full history with filtering and detail views
- **NFT Gallery** - Browse your NFT collections with floor prices and traits

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion + GSAP
- Recharts
- Solana Web3.js + Wallet Adapter
- Vite

## Getting Started

```bash
npm install
npm run dev
```
